# MOTIVATOR — Cold Brew Coffee

Интернет-магазин холодного кофе **MOTIVATOR** с каталогом, корзиной, оформлением заказов и админ-панелью.

## Требования

- [Node.js](https://nodejs.org/) 18+ (LTS)

## Быстрый старт

```bash
cd Projects/motivator-coffee
npm install
npm start
```

- **Магазин:** http://localhost:3000
- **Админка:** http://localhost:3000/admin

## Админ-панель

1. Откройте http://localhost:3000/admin
2. Введите токен из файла `.env` (по умолчанию: `motivator-admin-2026`)
3. Просматривайте заказы, меняйте статусы

### Статусы заказов

| Статус | Описание |
|--------|----------|
| Новый | Только что поступил |
| В обработке | Принят в работу |
| Отправлен | Передан в доставку |
| Завершён | Доставлен клиенту |
| Отменён | Отменён |

## Каталог

6 вкусов холодного кофе (250 мл) — продажа только упаковками по 12 бутылок:

- Гранат, Кизил, Апельсин, Вишня, Чёрная смородина, Малина
- Клиент сам выбирает состав упаковки (12 бутылок на выбор)
- Стоимость упаковки: **2 200 ₽**

## Настройка

Файл `.env`:

```
PORT=3000
ADMIN_TOKEN=ваш-секретный-токен
```

Заказы хранятся в JSON-файле (`data/orders.json`).

## Стек

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express
- **Хранение заказов:** JSON-файл
